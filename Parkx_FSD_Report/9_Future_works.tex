\chapter{FUTURE WORKS}
While ParkX provides a sturdy foundational architecture for smart parking, subsequent iterations can leverage emerging technologies to build a fully autonomous ecosystem.

\section{Recommended Enhancements}
\begin{enumerate}
    \item \textbf{IoT Sensor Integration:} 
    Deploying hardware sensors across parking slots to detect vehicle presence. This would enable real-time synchronization between the physical slot and the ParkX database without manual owner intervention, further automating the occupancy tracking.
    
    \item \textbf{Automated Payment Gateways:} 
    Integrating Stripe or Razorpay APIs to allow upfront digital payments during the booking phase, fully automating the financial transaction between Driver and Owner and reducing gate-side delays.
    
    \item \textbf{License Plate Recognition (ALPR):} 
    Utilizing machine learning cameras at entry barriers to scan driver plates and automatically lift gates if an active ParkX booking exists, providing a frictionless arrival experience.
    
    \item \textbf{Dynamic Fleet Pricing:} 
    Implementing an algorithmic pricing model that adjusts slot costs based on current venue capacity, peak traffic hours, or special events, maximizing revenue for property owners.
\end{enumerate}
