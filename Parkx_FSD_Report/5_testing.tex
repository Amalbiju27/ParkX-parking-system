\chapter{TESTING}

\section{Security \& Performance Verification}
Security testing ensures the platform remains resilient against unauthorized access and common web vulnerabilities.
\begin{itemize}
    \item \textbf{CSRF Protection:} Verified that all POST requests require a valid token; missing tokens trigger 419 errors.
    \item \textbf{Query Optimization:} Use of Eloquent with PostgreSQL row-level locking ensures rapid, conflict-free database responses under load.
\end{itemize}
